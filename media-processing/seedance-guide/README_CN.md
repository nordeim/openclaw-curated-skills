# 🎬 Seedance 2.0 分镜向导 (Seedance Storyboard Director)

[English](README.md) | [中文](README_CN.md)

专为 **即梦 (Jimeng) Seedance 2.0** 打造的专业分镜与提示词向导。本技能旨在帮助用户将简单的创意火花，转化为“电影级”的视频生成提示词，充分释放 Seedance 的多模态能力。

## ✨ 核心亮点

-   **🎬 分镜驱动工作流**：将您的视频创意拆解为专业的时间轴（0-3s, 3-6s 等），让画面更有层次感。
-   **🛠️ 多模态深度优化**：精通 `@图片`、`@视频`、`@音频` 引用语法，确角色一致性与动作复刻。
-   **🎥 专家级镜头语言**：内置丰富的电影拍摄语言（如希区柯克变焦、推拉摇移、长镜头等）。
-   **🩹 视频延长与编辑**：提供精准的指令引导，用于平滑延长视频或“颠覆”已有剧情。
-   **🏮 丰富场景模版**：内置情感叙事、商业广告、武侠打斗、以及除夕团圆等节日场景模版。

## 🚀 使用方法

直接输入您的创意构想即可唤醒技能：

> “/seedance-guide 一个温暖的除夕夜团圆饭视频。”

技能将作为您的 **创意导演**，通过询问风格、镜头要求及可用素材，一步步引导您生成最终的专业提示词。

## 📁 文件结构

-   `SKILL.md`：技能核心逻辑与技术指令。
-   `usage_and_examples.md`：丰富的场景化提示词模版库。

## 📝 Seedance 2.0 基础知识

Seedance 2.0 支持最多 **12 个文件** 作为输入：
-   **图片 (≤9)**：用于角色参考、风格参考、首/尾帧。
-   **视频 (≤3)**：用于动作参考、运镜参考、视频延长 (2-15s)。
-   **音频 (≤3)**：用于背景音乐或对口型参考。
