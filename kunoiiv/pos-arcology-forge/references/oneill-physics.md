# O'Neill Cylinder Physics Reference

## Core Formulas
- **Centripetal Gravity**: g = ω² r  →  ω = √(g / r)  (g=0.21 m/s² target)
- **Volume (Cyl)**: π r² L  (L=10km baseline)
- **Pop Density**: 1 person / 100 m³ habitat (conservative; scalable w/ replicators)
- **Power Needs**: 1kW/person min (fusion baseline)

## Stability Metrics
- Orbital decay <1%/yr → min delta-v budget
- Coriolis effects: r > 1km to minimize
