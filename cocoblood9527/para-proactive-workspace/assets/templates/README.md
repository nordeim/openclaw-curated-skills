# Workspace

Personal workspace combining **PARA Method** + **Proactive Agent Architecture**

## üìÅ Directory Structure

### PARA Method (Content Organization)

| Directory | Purpose | Example Contents |
|-----------|---------|------------------|
| `1-projects/` | Active projects with deadlines and goals | Client work, product launches, events |
| `2-areas/` | Ongoing responsibilities without deadlines | Health, finance, learning, relationships |
| `3-resources/` | Reference materials and notes | Articles, books, templates, code snippets |
| `4-archives/` | Completed or paused projects/areas | Past projects, old notes, inactive areas |
| `+inbox/` | Temporary inbox (review weekly) | Unsorted items, quick captures |
| `+temp/` | Scratch space | Drafts, experiments, cache |

### Proactive Agent Architecture (AI Memory)

| Directory/File | Purpose |
|----------------|---------|
| `memory/` | Daily logs and working buffer |
| `.learnings/` | Error logs, learnings, feature requests |
| `ONBOARDING.md` | First-run setup and user profile |
| `AGENTS.md` | Operating rules and workflows |
| `SOUL.md` | Agent identity and principles |
| `USER.md` | Human context and preferences |
| `MEMORY.md` | Curated long-term memory |
| `SESSION-STATE.md` | Active working memory (current task) |
| `HEARTBEAT.md` | Periodic self-improvement checklist |
| `TOOLS.md` | Tool configurations and notes |
| `.agents/` | Agent-specific configurations |

### Other

| Directory | Purpose |
|-----------|---------|
| `skills/` | Installed agent skills |
| `.clawhub/` | Skill package manager data |

## üöÄ Quick Start

### For You (Human)

1. **Capture** ‚Üí Drop files into `+inbox/`
2. **Process** ‚Üí Sort inbox items into PARA folders
3. **Review** ‚Üí Weekly review of `+inbox/` and `2-areas/`

### For Your Agent

1. **Read on startup** ‚Üí `SOUL.md`, `USER.md`, `SESSION-STATE.md`
2. **Log continuously** ‚Üí `memory/`, `.learnings/`
3. **Review periodically** ‚Üí `HEARTBEAT.md` checklist

---

## üîÑ Workflows

### New Project
```
Create folder in 1-projects/ ‚Üí Work on it ‚Üí Move to 4-archives/ when done
```

### Daily Notes
```
Agent writes to memory/YYYY-MM-DD.md ‚Üí Periodically distill to MEMORY.md
```

### Learning Capture
```
Error occurs ‚Üí Log to .learnings/ERRORS.md ‚Üí Fix ‚Üí Promote to AGENTS.md if valuable
```

## üìù Maintenance

- **Daily**: Clear `+temp/`, process `+inbox/`
- **Weekly**: Review `2-areas/`, archive completed projects
- **Monthly**: Clean `4-archives/`, review `.learnings/`
