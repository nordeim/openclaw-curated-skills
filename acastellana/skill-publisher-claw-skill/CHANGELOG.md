# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-30

### Added
- Initial release
- `audit.sh` - Comprehensive skill auditing
- `fix.sh` - Auto-fix common issues
- `scaffold.sh` - Create new skills from templates
- `publish.sh` - One-command publish workflow
- `analyze.sh` - Size and token analysis
- `validate-links.sh` - Link checker
- `score.sh` - Quality scoring (0-100)
- `changelog.sh` - Generate changelog from commits
- Templates for SKILL.md, README.md, LICENSE, .gitignore
- Pre-commit hook for security checks
- GitHub Actions workflow template
- Documentation for versioning and deprecation
