# 热梗文档格式

每条热梗写入 Dify 时使用以下 Markdown 结构：

```markdown
# 梗名称

## 基本信息
- 梗类型：音频梗/视频梗/流行语/梗图/表情包
- 热度等级：🔥🔥🔥/🔥🔥/🔥
- 流行时间：2025上半年
- 二创潜力：高/中/低
- 平台来源：抖音、B站、微博

## 含义解释
详细解释这个梗的含义、背景和为什么火...

## 金句台词
"经典台词1" "经典台词2"

## 出处来源
来源说明（视频、歌曲、事件等）

## 情感标签
搞笑、自嘲、无奈、励志、调侃、荒诞、温暖、讽刺

## 适用场景
职场、日常、学习、社交、美食、旅行

## 使用场景示例
- 具体使用示例1
- 具体使用示例2
- 具体使用示例3

## 剧本融入指南
这个梗如何自然融入漫剧剧本对话，适合什么角色说，什么场景用，怎么制造喜剧效果。
```

## 字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| 梗名称 | ✅ | 简短、易识别，也是 Dify 文档的 `name` |
| 梗类型 | ✅ | 音频梗/视频梗/流行语/梗图/表情包 |
| 热度等级 | ✅ | 🔥🔥🔥 顶级 / 🔥🔥 高热 / 🔥 普通 |
| 流行时间 | ✅ | 如 "2025上半年"、"2025下半年" |
| 二创潜力 | ✅ | 高/中/低 |
| 平台来源 | ✅ | 主要传播平台 |
| 含义解释 | ✅ | 2-4句话解释清楚 |
| 金句台词 | ✅ | 最经典的台词/用法 |
| 出处来源 | ✅ | 梗的起源 |
| 情感标签 | ✅ | 用顿号分隔 |
| 适用场景 | ✅ | 用顿号分隔 |
| 使用场景示例 | ✅ | 3个具体示例 |
| 剧本融入指南 | ✅ | 如何用在漫剧剧本中 |

## 热度评估标准

- 🔥🔥🔥 **顶级热梗**：全网传播，各平台都有大量二创，持续时间长
- 🔥🔥 **高热梗**：在特定平台或圈层广泛传播，有一定二创
- 🔥 **普通热梗**：有一定热度但传播范围有限
