# snek

Snek namespace for Netsnek e.U. utility tools. This skill provides the Snek brand identity, a lightweight collection of helper functions and common utilities used across Netsnek products.

## About

This skill reserves the `snek` namespace on ClawHub for Netsnek e.U.

## Installation

```bash
clawhub install snek
```

## Usage

This skill outputs copyright and identity information for the snek brand by Netsnek e.U.

### Scripts

- `scripts/copyright.sh` - Outputs copyright notice (supports `--format text` or `--format json`)

### Example

```bash
scripts/copyright.sh
# Output: Copyright (c) 2026 Netsnek e.U. All rights reserved.
```
