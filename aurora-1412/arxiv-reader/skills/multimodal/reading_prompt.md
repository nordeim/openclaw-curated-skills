# Multimodal Learning 论文阅读指南

阅读此类论文时，请特别关注以下方面：

1. **模态对齐方式**：不同模态如何映射到同一表征空间？使用了什么对齐目标？
2. **模型架构**：视觉编码器、语言模型、连接模块分别是什么？如何融合？
3. **训练策略**：是否分阶段训练？预训练和微调的数据分别是什么？
4. **生成质量**（如涉及生成任务）：生成结果的质量如何？有哪些控制手段？
5. **评估指标**：使用了哪些多模态 Benchmark？在各项任务上的表现？
6. **数据构建**：多模态训练数据如何收集和清洗？
7. **效率与部署**：模型大小、推理速度、部署方案？

请在生成笔记时使用以下结构：
- **一句话总结**
- **核心架构**（各模态组件及其交互方式）
- **训练流程**
- **关键实验结果**
- **创新点**
- **局限性**
