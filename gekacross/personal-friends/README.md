# Personal Friends

Помощник по друзьям и социальной жизни для OpenClaw. Помнит кто есть кто, дни рождения, долги, планы.

## Что умеет

- Хранить профили друзей (имя, ДР, интересы, как познакомились)
- Трекать долги (кто кому и сколько должен)
- Записывать планы встреч
- Напоминать о днях рождения
- Хранить идеи подарков

## Команды

```
добавь друга Саша          — создать профиль друга
др Саши 15 марта           — записать день рождения
Саша должен мне 500        — записать долг
я должен Саше 200          — записать долг
план встречи с Сашей       — записать встречу
все друзья                 — показать список
долги                      — показать все долги
инструкция                 — показать все команды
```

## Установка

```bash
clawdhub install personal-friends
```

## Данные

Хранятся в `knowledge/personal/friends.md` в воркспейсе агента.
