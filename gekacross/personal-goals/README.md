# Personal Goals

Партнёр по целям для OpenClaw. Помогает ставить, декомпозировать и трекать цели без занудства.

## Что умеет

- Ставить цели с дедлайнами
- Декомпозировать большие цели на шаги
- Трекать прогресс в процентах
- Отмечать выполненные шаги
- Делать ретроспективу
- Категоризировать (здоровье, финансы, карьера, личное)

## Команды

```
новая цель: [название]     — поставить цель
прогресс [цель] 50%        — обновить прогресс
все цели                   — показать список
шаги для [цель]            — декомпозировать
выполнен шаг [шаг]         — отметить сделанным
дедлайн [цель] 1 июня      — установить дедлайн
заморозить [цель]          — поставить на паузу
инструкция                 — показать все команды
```

## Установка

```bash
clawdhub install personal-goals
```

## Данные

Хранятся в `knowledge/personal/goals.md` в воркспейсе агента.
