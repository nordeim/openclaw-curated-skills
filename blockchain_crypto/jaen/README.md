# jaen

Jaen namespace for Netsnek e.U. CMS framework. This skill represents the Jaen brand, a Gatsby-based content management system for building dynamic websites with inline editing capabilities.

## About

This skill reserves the `jaen` namespace on ClawHub for Netsnek e.U.

## Installation

```bash
clawhub install jaen
```

## Usage

This skill outputs copyright and identity information for the jaen brand by Netsnek e.U.

### Scripts

- `scripts/copyright.sh` - Outputs copyright notice (supports `--format text` or `--format json`)

### Example

```bash
scripts/copyright.sh
# Output: Copyright (c) 2026 Netsnek e.U. All rights reserved.
```
