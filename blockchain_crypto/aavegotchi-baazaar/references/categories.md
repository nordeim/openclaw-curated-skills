# Category Maps

These category integers are required when creating listings.

## ERC721 categories

- `0` PORTAL_CLOSED
- `1` PORTAL_SUMMONING
- `2` PORTAL_OPEN
- `3` AAVEGOTCHI
- `4` REALM/PARCEL
- `5` FAKEGOTCHIS

## ERC1155 categories

- `0` WEARABLE
- `1` BADGE
- `2` CONSUMABLE
- `3` TICKET
- `4` INSTALLATION
- `5` TILE
- `6` FAKECARD
- `7` ALLOY
- `8` SCHEMATIC
- `9` GEODE
- `10` ESSENCE
- `11` CORE
- `12` GUARDIAN_SKINS
- `13` GUARDIANS_PROFILE
