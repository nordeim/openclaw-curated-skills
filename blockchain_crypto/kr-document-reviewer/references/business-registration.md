# 사업자등록증 검토 체크리스트

지원기업/공급기관 모두 동일한 형식으로 검토한다.

## 필수 기재 항목

| # | 항목 | 검증 규칙 |
|---|------|-----------|
| 1 | 등록번호 | 사업자등록번호 10자리, XXX-XX-XXXXX |
| 2 | 상호(법인명) | 비어있지 않을 것 |
| 3 | 대표자 | 비어있지 않을 것 |
| 4 | 개업년월일 | 유효한 날짜 |
| 5 | 법인등록번호 | 법인인 경우 13자리 |
| 6 | 사업장 소재지 | 비어있지 않을 것 |
| 7 | 본점 소재지 | 법인인 경우 기재 |
| 8 | 사업의 종류 (업태) | 비어있지 않을 것 |
| 9 | 사업의 종류 (종목) | 비어있지 않을 것 |
| 10 | 교부일/발급일 | 유효한 날짜 |
| 11 | 관할 세무서장 | 기재 여부 |

## 검증 규칙

### 사업자등록번호 검증
- 10자리 숫자
- 체크섬: 가중치 [1,3,7,1,3,7,1,3,5] 적용 후 mod 10

### 법인등록번호 검증
- 13자리 숫자 (XXXXXX-XXXXXXX)
- 앞 6자리: 등기소 코드 + 법인 종류

### 유효성 확인
- 발급일이 최근인지 (보통 3개월 이내 권장)
- 휴업/폐업 여부 확인 불가 시 유의사항 기재
- 국세청 홈택스에서 사업자 상태 조회 권장

## 교차 검증
- 상호/대표자명 ↔ 계약서 당사자 정보 일치
- 사업자등록번호 ↔ 세금계산서/거래명세서/견적서 번호 일치
- 사업장 소재지 ↔ 세금계산서 주소 일치
- 업태/종목 ↔ 세금계산서 업태/종목 일치

## 흔한 오류
- 오래된 사업자등록증 제출 (주소 변경, 대표자 변경 반영 안 됨)
- 세금계산서와 사업자등록번호 불일치
- 휴/폐업 사업자 등록증 제출
