# 사업비 요청 공문 검토 체크리스트

## 필수 기재 항목

| # | 항목 | 검증 규칙 |
|---|------|-----------|
| 1 | 문서번호 | 비어있지 않을 것 |
| 2 | 발신 | 요청 기관/업체명 |
| 3 | 수신 | 지원/관리 기관명 |
| 4 | 제목 | 사업비 요청 관련 제목 |
| 5 | 사업명 | 구체적 사업/과제명 |
| 6 | 요청 금액 | 숫자 + 한글 병기 권장 |
| 7 | 요청 내역 | 세부 항목별 금액 |
| 8 | 입금 계좌 정보 | 은행, 계좌번호, 예금주 |
| 9 | 첨부서류 목록 | 제출 서류 리스트 |
| 10 | 작성일자 | 유효한 날짜 |
| 11 | 직인/서명 | 발신기관 직인 또는 대표자 서명 |

## 검증 규칙

### 금액 검증
- 요청 금액 합계가 세부 내역 합산과 일치하는지
- 부가세 포함/별도 명시 여부
- 숫자 금액과 한글 금액 일치 여부

### 계좌 정보
- 통장사본의 계좌번호와 일치하는지
- 예금주가 요청 기관명과 일치하는지

### 첨부서류
- 공문에 명시된 첨부서류가 실제로 모두 제출되었는지

## 교차 검증
- 요청 금액 ↔ 계약서 계약금액 일치
- 요청 금액 ↔ 세금계산서 합계 일치
- 입금 계좌 ↔ 통장사본 정보 일치
- 사업명 ↔ 계약서 계약 목적 일치

## 흔한 오류
- 요청 금액과 세금계산서 금액 불일치
- 입금 계좌 정보와 통장사본 불일치
- 첨부서류 목록에 있으나 실제 미제출
- 직인/서명 누락
