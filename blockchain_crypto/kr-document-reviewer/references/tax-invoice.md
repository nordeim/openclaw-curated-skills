# 세금계산서 검토 체크리스트

## 필수 기재 항목 (부가가치세법 제32조)

| # | 항목 | 검증 규칙 |
|---|------|-----------|
| 1 | 공급자 등록번호 | 사업자등록번호 10자리, XXX-XX-XXXXX 형식 |
| 2 | 공급자 상호 | 비어있지 않을 것 |
| 3 | 공급자 성명(대표자) | 비어있지 않을 것 |
| 4 | 공급자 사업장 소재지 | 비어있지 않을 것 |
| 5 | 공급받는자 등록번호 | 사업자등록번호 10자리 |
| 6 | 공급받는자 상호 | 비어있지 않을 것 |
| 7 | 공급가액 | 숫자, 0 이상 |
| 8 | 세액 | 공급가액 × 10% (일반과세) |
| 9 | 작성일자 | 유효한 날짜, 미래 날짜 아닐 것 |
| 10 | 품목 | 1개 이상 기재 |

## 검증 규칙

### 사업자등록번호 검증
- 10자리 숫자
- 체크섬 검증: 가중치 [1,3,7,1,3,7,1,3,5] 적용 후 mod 10
- 국세청 진위확인 권장 (hometax.go.kr)

### 세액 검증
- 일반과세: 세액 = 공급가액 × 10%
- 영세율: 세액 = 0
- 합계금액 = 공급가액 + 세액

### 전자세금계산서 추가 확인
- 승인번호 24자리 존재 여부
- 국세청 전송 여부
- 발급일과 전송일 차이 (익월 11일까지)

## 흔한 오류
- 세액 계산 불일치 (반올림 차이)
- 사업자등록번호 오기재
- 작성일자와 실제 거래일 불일치
- 공급가액과 세액 위치 바뀜
