# Briefing Templates

## Morning Briefing (08:30 Weekdays)
1. **Market Overview**: S&P 500, Nasdaq, BIST 100, BTC, ETH, USD/TRY, EUR/USD, Gold
2. **Portfolio Summary**: Each position with current price and daily change
3. **Active Alerts**: Count of active price alerts
4. **Key Events**: Earnings reports today, economic data releases

## Market Close (23:00 Weekdays)
1. **Closing Prices**: All major indices and tracked assets
2. **Daily P&L**: Portfolio performance for the day
3. **Top Movers**: Biggest gainers/losers in portfolio

## Weekend Recap (Saturday 10:00)
1. **Crypto 24/7**: BTC, ETH, SOL, BNB with 7-day performance
2. **Forex**: USD/TRY, EUR/USD weekend rates
3. **Weekly Performance**: Portfolio weekly returns
4. **Upcoming Week**: Key events, earnings, economic releases

## Formatting Guidelines
- Use bold for section headers
- Use emoji for sentiment (green/red circles)
- Keep each briefing under 2000 characters for Telegram readability
- Include percentage changes for all prices
- Round to 2 decimal places for stocks, 4 for forex
