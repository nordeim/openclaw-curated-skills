# CRM Manager

A lightweight CSV-based CRM. Add contacts, track pipeline stages, get follow-up reminders. No Salesforce required.

## What It Does

- Manages contacts in a local CSV file
- Tracks deals through 8 pipeline stages
- Flags overdue follow-ups and stale deals
- Shows pipeline summaries with deal values
- Includes a ready-to-use CSV template
