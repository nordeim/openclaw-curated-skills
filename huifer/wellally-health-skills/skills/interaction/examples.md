# interaction 技能示例

## 示例 1: 检查相互作用（发现严重问题）

```
用户: 检查药物相互作用

输出:
🚨 药物相互作用检查

检测到 3 个相互作用：

🆘 绝对禁忌（1个）
1. 华法林 + 阿司匹林
   严重程度：🆘 绝对禁忌 (X类)
   风险：显著增加出血风险
   建议：避免合用，除非在医生监测下

🔴 禁忌（1个）
2. 辛伐他汀 + 氨氯地平
   严重程度：🔴 禁忌 (D类)
   风险：增加肌肉毒性风险

🟠 相对禁忌（1个）
3. 华法林 + 富含维生素K的食物
   严重程度：🟠 相对禁忌 (C类)
   风险：降低抗凝效果
```

## 示例 2: 检查相互作用（无问题）

```
用户: 检查药物相互作用

输出:
✅ 药物相互作用检查

检查结果：
✅ 未检测到药物相互作用
✅ 未检测到药物-疾病冲突
✅ 未检测到剂量问题
✅ 未检测到饮食冲突

您的用药方案安全性良好
```

## 示例 3: 列出规则库

```
用户: 列出相互作用规则

输出:
📚 药物相互作用规则库

统计信息：
总规则数：80 条
🟢 A类（安全）：10 条
🟡 B类（谨慎）：15 条
🟠 C类（相对禁忌）：30 条
🔴 D类（禁忌）：20 条
🆘 X类（绝对禁忌）：5 条
```

## 示例 4: 列出特定级别规则

```
用户: 列出 X 类规则

输出:
🆘 绝对禁忌规则（X类）：

1. 华法林 + 阿司匹林
   风险：严重出血
   建议：避免合用

2. MAO抑制剂 + SSRIs
   风险：5-羟色胺综合征
   建议：严格禁止
```

```
输出:

药物1 + 药物2

严重程度：🟠 相对禁忌 (C类)
风险描述：增加乳酸酸中毒风险
```
