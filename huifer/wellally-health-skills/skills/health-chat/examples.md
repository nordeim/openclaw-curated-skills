# Health Chat Skill Examples

## 示例 1：日常健康询问

```
用户: /health-chat 今天感觉有点累，是什么原因？

处理流程：
1. 读取 data/profile.json 获取基本信息
2. 读取 data/health-feeling-logs.json 检查近期健康日志
3. 读取 data/sleep-tracker.json（如存在）检查睡眠质量
4. 分析可能的疲劳原因

回复示例：

## 📊 健康数据分析

基于您的健康数据，我发现了以下可能与疲劳相关的因素：

### 睡眠状况
- 近7天平均睡眠：6.2小时（建议7-9小时）
- 睡眠质量评分：65/100（偏低）
- 最近3次记录显示入睡困难

### 近期健康感受
- 昨天记录：感觉疲惫
- 前天记录：精力不足

## 🎯 可能的原因

1. **睡眠不足** - 平均睡眠时间低于建议值
2. **睡眠质量不佳** - 需要改善睡眠环境或习惯
3. **可能的相关因素** - 工作压力、运动不足

## 💡 建议

1. 今晚尝试提前30分钟入睡
2. 睡前1小时避免使用电子设备
3. 保持卧室安静、黑暗的环境
4. 如疲劳持续超过2周，建议就医检查

---

⚕️ 医疗免责声明：本系统提供的健康信息仅供参考，不能替代专业医疗建议。
如有健康问题，请咨询医疗专业人员。

后续建议问题：
- 我的睡眠数据详细情况是什么？
- 如何改善睡眠质量？
- 需要做哪些检查？
```

1. 读取 data/hypertension-tracker.json

5. 分析可能原因

回复示例：

## 🔍 症状分析：头晕

根据您的健康数据，我分析了以下可能原因：

### 📊 相关数据分析

1. **血压情况**
   - 最近测量：132/85 mmHg
   - 收缩压偏高可能导致头晕

2. **血糖情况**
   - 最近空腹血糖：5.8 mmol/L
   - 处于正常范围

3. **用药情况**
   - 正在服用：氨氯地平
   - 可能的副作用：头晕（尤其剂量调整时）

4. **睡眠情况**
   - 平均睡眠：6.2小时
   - 睡眠不足也可能导致头晕

### 🎯 可能的原因

| 可能性 | 原因 | 说明 |
|--------|------|------|
| 🔴 高 | 血压偏高 | 收缩压132mmHg可能引起头晕 |
| 🟡 中 | 药物副作用 | 氨氯地平可能导致头晕 |
| 🟡 中 | 睡眠不足 | 睡眠时间偏少 |
| 🟢 低 | 血糖异常 | 目前血糖正常 |

## 💡 建议

1. **立即测量血压**，确认当前血压水平
2. **保证充足休息**，今晚提前睡眠

   - 胸痛或胸闷

5. **建议就诊科室**：神经内科或心内科

⚕️ 医疗免责声明：本系统提供的健康信息仅供参考，不能替代专业医疗建议。

## 示例 7：数据查询 - 趋势分析

```
用户: /health-chat 我的体重最近有什么变化？

处理流程：
1. 搜索所有可能记录体重的数据文件
2. 提取体重记录
3. 计算趋势和变化

回复示例：

## 📈 体重趋势分析

### 数据概览
| 时间段 | 平均体重 | 变化 |
|--------|---------|------|
| 近1周 | 73.2kg | +0.3kg |
| 近1月 | 73.0kg | +0.8kg |
| 近3月 | 72.6kg | +1.2kg |

### 趋势图（文字）
```
75kg ┤                    ╭─ 73.2
74kg ┤              ╭────╯
73kg ┤         ╭────╯
72kg ┤    ╭───╯      72.0
71kg ┤ ╭──╯
70kg ─┴───────────────────────
     11月  12月  1月  2月
```

### 分析结论
- **总体趋势**：📈 上升（近3个月+1.2kg）
- **变化速度**：约0.4kg/月
- **BMI变化**：23.8 → 24.9

## 🎯 建议

1. **控制饮食热量**：减少约200千卡/天
2. **增加运动量**：每周增加2次有氧运动
3. **规律称重**：每周固定时间称重
4. **记录饮食**：有助于发现热量摄入问题

目标：3个月内减重2kg，恢复至72kg以下

---

⚕️ 医疗免责声明：本系统提供的健康信息仅供参考。
```
