```
类型: 减重目标
标题: 6个月内减重5公斤
开始日期: 2025-01-01
目标日期: 2025-07-01

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
SMART 原则验证:
✅ 具体: 减重5公斤
✅ 可衡量: 可通过体重秤测量
✅ 可实现: 平均每月减重0.83kg (健康范围)
✅ 相关: 与健康管理相关

• 每周记录体重
• 控制每日热量摄入
• 规律运动，每周至少3次
```

```
用户: /goal habit set 每天早上7点拉伸10分钟

AI: ✅ 健康习惯已设定

习惯详情:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
习惯ID: habit_20250101_000001
名称: 晨间拉伸
类型: 每日习惯
触发时间: 每天07:00
持续时间: 10分钟

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• 从小习惯开始，建立一致性

• 偶尔错过没关系，不要连续错过
```
