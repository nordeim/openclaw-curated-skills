# 儿童疫苗接种示例

## 示例1: 记录疫苗接种

### 用户输入
```
/child-vaccine record 乙肝疫苗 第1针 completed 2020-01-01
```

### 系统输出
```
疫苗接种已记录

疫苗信息：
  疫苗：乙肝疫苗
  剂次：第1针
  接种日期：2020年1月1日
  类别：一类疫苗

接种进度：
  已完成：1/3剂次

下次接种：
  2020年2月1日（1月后）
  乙肝疫苗 第2针

重要提示：
本系统仅供疫苗接种记录，不能替代专业医疗建议。

所有接种请在预防接种门诊进行，
接种前请告知医生儿童健康状况。

数据已保存
```

## 示例3: 查看待接种疫苗

### 用户输入
```
/child-vaccine due
```

### 系统输出
```
待接种疫苗

儿童：小明（5岁5个月）

未来30天内需要接种：

  2025-08-01（还有42天）
    百白破疫苗 第5针
    流脑A+C群疫苗 第2针

建议：
  请提前预约接种时间
  接种前请确保儿童身体状况良好
  如有发热、急性疾病等情况请延期接种

数据已保存
```

## 示例4: 查看逾期未种


## 示例5: 记录不良反应

### 用户输入
```
/child-vaccine reaction 百白破 发热 轻度
```

### 系统输出
```
不良反应已记录

反应信息：
  疫苗：百白破疫苗
  记录日期：2025年1月14日
  症状：发热
  严重程度：轻度

处理建议：
  轻度发热是正常反应
  可物理降温
  多喝水
  观察24小时

何时就医：
  发热超过39持续24小时
  伴有严重哭闹
  局部红肿严重
  出现过敏反应

数据已保存
```
