# save-report 技能示例

## 示例 1: 保存生化检查（自动提取日期）

```
用户: @医疗报告/血液检查.jpg

分析图片 -> 提取数据

保存:
{
  "type": "生化检查",
  "date": "2025-10-07",
  "hospital": "XX医院",
  "items": [
    {"name": "白细胞计数", "value": "6.5", "unit": "×10^9/L", "is_abnormal": false},
    {"name": "血红蛋白", "value": "145", "unit": "g/L", "is_abnormal": false}
  ]
}

输出:
✅ 检查单保存成功！
类型：生化检查（血液常规）
日期：2025-10-07（从图片提取）
提取到 15 项检查指标
文件路径：data/生化检查/2025-10/2025-10-07_血液常规.json
```

## 示例 2: 手动指定日期

```
用户: @医疗报告/血液检查.jpg 2025-10-07

保存:
{
  "date": "2025-10-07",  // 使用用户指定日期
  ...
}

输出:
✅ 检查单保存成功！
日期：2025-10-07（使用用户指定日期）
```

## 示例 5: 检测到异常指标

```
用户: @医疗报告/生化检查.jpg

分析结果：
检测到 3 项异常指标：
- 尿酸: 486 μmol/L ❌ (参考: 150-420)
- 总胆固醇: 6.2 mmol/L ❌ (参考: <5.2)
- 低密度脂蛋白: 3.5 mmol/L ❌ (参考: <3.4)

保存时标记 is_abnormal: true
```
