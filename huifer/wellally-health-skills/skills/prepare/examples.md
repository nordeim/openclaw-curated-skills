# Prepare 技能示例

## 示例 1: 通用准备指南 (无参数)

```
用户: /prepare

AI: 🏥 医院就诊准备指南

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 就诊准备清单
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

必备证件:
☐ 身份证/医保卡 (必带)
☐ 医保卡/就诊卡 (必带)
☐ 银行卡或手机支付
☐ 既往病历资料 (如有)

资料准备:
☐ 最近的检查报告和影像资料
☐ 用药清单或正在服用的药物
☐ 既往出院小结 (如有)
☐ 过敏史清单 (必带) ⭐
☐ 过敏急救药物 (如严重过敏) ⭐

就诊流程:
1. 预约挂号 (建议提前预约)
2. 提前15-30分钟到达医院
3. 自机取号或签到
4. 候诊区等待叫号
5. 医生问诊和检查
6. 缴费、做检查或取药
7. 等结果并复诊 (如需要)

就诊技巧:
✓ 说清主要症状: 哪里不舒服、多久了
✓ 讲明病史: 既往病史、手术史、过敏史
✓ 列出用药: 正在服用的药物
✓ 提出疑问: 准备好想问的问题
✓ 理解医嘱: 不清楚的及时询问

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
您的健康摘要:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

基本信息:
• 年龄: 45岁
• 血型: A型

过敏史重点提示 (3条):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔴 青霉素 - 严重过敏 (必须告知)
🔴 碘造影剂 - 严重过敏 (检查时提醒)
🆘 蜜蜂叮咬 - 过敏性休克 (携带急救药)

⚠️ 就诊时必须主动告知医护人员！

最近症状 (7天内):
• 12-30: 头痛 (轻度，2天)
• 12-28: 失眠 (轻度，1周)
• 12-25: 乏力 (轻度，3天)

正在用药:
• 阿司匹林 100mg 每天1次 (早餐后)
• 氨氯地平 5mg 每天2次 (早晚各一次)
• 二甲双胍 500mg 每天3次 (餐后)

最近检查 (30天内):
• 12-20: 头颅CT - 未见明显异常
• 12-15: 血常规 - 白细胞轻度升高
• 12-10: 生化全项 - 空腹血糖 6.8 mmol/L↑

最近诊断:
• 11-10: 高血压病 (2级，中危)
• 10-05: 紧张性头痛
• 09-20: 2型糖尿病

待复查项目:
• 2026-01-15: 神经内科复查
• 2026-01-10: 血糖监测

就诊建议:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📊 健康状况评估: 您有慢性疾病 (高血压、糖尿病)，建议定期复查

💊 用药提醒:
   - 正在服用3种药物，请按时服药
   - 如出现药物不良反应，及时告知医生

⚠️ 需要关注:
   - 头痛持续2天未缓解，建议就医
   - 空腹血糖偏高，建议内分泌科复查

📅 就诊科室建议:
   根据您的症状和病史，建议就诊科室:
   • 神经内科 - 头痛症状
   • 内分泌科 - 血糖管理
   • 心内科 - 高血压随访

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💡 提示: 这份健康摘要可以在就诊时向医生展示，帮助医生快速了解您的健康状况！

祝您健康！
```

```
用户: /prepare 头疼

AI: 🏥 就诊准备指南

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
就诊信息:
症状: 头疼
推荐科室: 神经内科
就诊日期: 今天

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
科室简介:
神经内科主要诊治脑血管疾病、头痛、
头晕、失眠、癫痫等神经系统疾病。

必备证件:
☐ 身份证/医保卡 (必带)
☐ 医保卡/就诊卡 (必带)
☐ 银行卡或手机支付
☐ 既往病历资料 (如有)

资料准备:
☐ 最近检查报告和影像资料
☐ 用药清单或正在服用的药物
☐ 既往出院小结
☐ 过敏史清单 (必带) ⭐
☐ 过敏急救药物 (如携带) ⭐

注意事项:
• 建议提前预约挂号
• 就诊前保证充足休息
• 穿宽松衣物，方便检查
• 如测量血压，就诊前30分钟避免剧烈运动

您的健康数据摘要:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[健康摘要内容...]

💡 就诊提示:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
就诊时请向医生提供:
1. 主要症状: 头疼2天，以胀痛为主
2. 既往病史: 高血压2级，中危组
3. 正在用药: 阿司匹林、氨氯地平
4. 过敏史: 青霉素

建议重点关注:
• 头疼与高血压的关系
• 当前用药是否需要调整
• 是否需要进一步检查 (如头颅MRI)

紧急情况:
如出现以下情况，请立即就医:
• 剧烈头痛，不同于以往
• 伴有发热、颈部僵硬
• 出现视力模糊、意识改变
• 肢体无力或麻木
```
