# radiation-data 技能示例

## 示例 1: 查询CT剂量

```
调用: 获取CT胸部剂量

返回: 7 mSv
```

## 示例 2: 计算调整后剂量

```
用户体表面积: 1.85 m²
标准剂量: 7 mSv

计算:
调整系数 = 1.85 / 1.73 = 1.07
调整后剂量 = 7 × 1.07 = 7.5 mSv
```

## 示例 3: 辐射衰减计算

```
初始剂量: 7 mSv
经过时间: 1年

计算:
残留剂量 = 7 × (0.5)^1 = 3.5 mSv

经过2年:
残留剂量 = 7 × (0.5)^2 = 1.75 mSv
```
