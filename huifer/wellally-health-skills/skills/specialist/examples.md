# Specialist 技能示例

## 示例 1: 心内科分析

```
用户: /specialist cardio recent 5

AI: 🏥 心内科专家分析中...

读取高血压管理数据...
读取最近5条检查记录...

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
心内科分析报告
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### 慢性病管理情况
**高血压控制状态**: 控制不佳
• 诊断时间: 2024-05-15
• 分级: 2级高血压
• 近期控制指标:
  - 平均血压: 145/95 mmHg
  - 目标: <140/90 mmHg
  - 达标率: 45% (不达标)
• 靎器官损害: 左心室肥厚
• 风险评估: 高危

### 近期检查数据
• 血脂异常: 总胆固醇 6.8 mmol/L↑, LDL-C 4.2 mmol/L↑
• BNP升高: 450 pg/mL↑ (提示心功能不全)
• 心电图: 左心室高电压

### 综合评估
您的高血压控制不理想，存在多重心血管风险因素:
1. 血压持续高于目标值
2. 血脂明显异常
3. BNP升高提示可能存在心功能不全
4. 已有左心室肥厚（靶器官损害）

### 建议
**生活方式:**
• 低盐低脂饮食，每日盐摄入<5g
• 规律有氧运动，每周150分钟
• 控制体重，BMI保持在18.5-24
• 戒烟限酒

**饮食调整:**
• 增加蔬菜水果摄入
• 选择低脂乳制品
• 减少饱和脂肪摄入
• 增加全谷物

**就医建议:**
⚠️ 建议尽快就诊心内科
• 调整降压药物方案
• 启动他汀类药物治疗
• 评估心功能
• 考虑做心脏彩超

**监测建议:**
• 每周测量血压2-3次
• 记录血压日记
• 1个月后复查血脂

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
重要免责声明: 本分析仅供参考，不能替代专业医疗诊断和治疗。所有医疗决策请遵从医生指导。
```

## 示例 2: 内分泌科分析

```
用户: /specialist endo all

AI: 🏥 内分泌科专家分析中...

读取糖尿病管理数据...
读取所有检查记录...

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
内分泌科分析报告
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### 慢性病管理情况
**2型糖尿病控制状态**: 控制不佳
• 诊断时间: 2023-08-10
• 分类: 2型糖尿病
• 近期控制指标:
  - 糖化血红蛋白: 7.5% (目标<7.0%)
  - 空腹血糖平均: 7.8 mmol/L
  - 餐后血糖平均: 11.2 mmol/L
  - 达标率: 约60%
• 并发症状态: 早期视网膜病变
• 风险评估: 中高危

### 近期检查数据
• 甲状腺功能: 正常
• 血脂异常: TG 2.8 mmol/L↑
• 尿微量白蛋白: 正常

### 综合评估
您的糖尿病控制需要加强:
1. 糖化血红蛋白超标，提示血糖控制不理想
2. 餐后血糖偏高明显
3. 已存在早期并发症（视网膜病变）
4. 合并血脂异常

### 建议
**生活方式:**
• 规律饮食，定时定量
• 餐后运动30分钟
• 控制碳水化合物摄入
• 减轻体重5-10%

**饮食调整:**
• 选择低GI食物
• 增加膳食纤维
• 控制份量
• 避免含糖饮料

**就医建议:**
⚠️ 建议就诊内分泌科
• 调整降糖方案
• 考虑加用餐后血糖调节药物
• 眼科定期随访
• 3个月后复查糖化血红蛋白

**监测建议:**
• 每周监测空腹血糖3次
• 每周监测餐后血糖2次
• 每3个月检测糖化血红蛋白

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
重要免责声明: 本分析仅供参考，不能替代专业医疗诊断和治疗。所有医疗决策请遵从医生指导。
```

## 示例 4: 骨科分析

```
用户: /specialist ortho all

AI: 🏥 骨科专家分析中...

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
骨科分析报告
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### 检查数据
• 骨密度检查: T值 -2.5 (骨质疏松)
• X光检查: 腰椎退行性变
• 钙: 2.1 mmol/L (正常低限)
• 维生素D: 18 ng/mL (缺乏)

### 综合评估
您存在骨质疏松:
1. T值-2.5，符合骨质疏松诊断标准
2. 维生素D严重缺乏
3. 钙摄入不足
4. 已有腰椎退行性变

### 建议
**生活方式:**
• 增加负重运动
• 预防跌倒
• 戒烟限酒
• 适量日晒

**饮食调整:**
• 增加钙摄入 (1000-1200mg/天)
• 补充维生素D
• 增加蛋白质摄入
• 限制咖啡因

**就医建议:**
⚠️ 建议就诊骨科/内分泌科
• 启动骨质疏松治疗
• 补充钙剂和维生素D
• 考虑抗骨质疏松药物
• 1-2年后复查骨密度

**监测建议:**
• 每年检测骨密度
• 每3-6个月检测维生素D水平

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
重要免责声明: 本分析仅供参考，不能替代专业医疗诊断和治疗。所有医疗决策请遵从医生指导。
```

```
• 考虑补充铁剂和锌剂
• 3个月后复查血常规

• 每3个月测量身高体重
• 每年体检一次
```
