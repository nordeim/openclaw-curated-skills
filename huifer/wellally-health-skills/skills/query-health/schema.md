# 查询医疗记录 Schema

医疗记录查询的完整数据结构定义。

## 查询类型

| 类型 | 关键词 | 说明 |
|-----|-------|------|
| all | all, 全部 | 所有记录 |
| biochemical | biochemical, 生化 | 生化检查 |
| imaging | imaging, 影像 | 影像检查 |
| recent | recent, 最近 | 最近N条 |
| date | date, 日期 | 指定日期 |
| abnormal | abnormal, 异常 | 异常指标 |

## 生化检查显示字段

| 字段 | 说明 |
|-----|------|
| 检查项目 | 项目名称 |
| 数值 | 检查结果 |
| 单位 | 测量单位 |
| 参考范围 | 正常值范围 |
| 状态 | ✅正常 / ❌异常 |

## 影像检查显示字段

| 字段 | 说明 |
|-----|------|
| 检查类型 | B超/CT/MRI/X光等 |
| 检查部位 | 检查区域 |
| 检查所见 | 详细描述 |
| 测量数据 | 尺寸数值 |
| 检查结论 | 诊断意见 |

## 数据存储

- 索引文件: `data/index.json`
- 生化检查: `data/生化检查/YYYY-MM/*.json`
- 影像检查: `data/影像检查/YYYY-MM/*.json`
