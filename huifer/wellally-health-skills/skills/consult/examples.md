| Abnormal blood glucose, abnormal thyroid function | Endocrinology |

| Abnormal kidney function, abnormal urinalysis | Nephrology |
| Abnormal complete blood count, abnormal coagulation | Hematology |
| Abnormal chest CT, abnormal infection markers | Pulmonology |
| Abnormal head imaging | Neurology |
| Abnormal tumor markers | Oncology |
| Multi-system abnormalities | General Practice |
