## 三、骨龄评估

### 示例 9: 正常骨龄
```
用户: /puberty bone-age 10.8

系统:
骨龄已记录

实际年龄：10.5岁
骨龄：10.8岁
差异：+0.3岁

评估：
骨龄与实际年龄基本一致，属正常范围。

差异解读：
• < -1岁：生长延迟
• -1 至 +1岁：正常范围
• > +1岁：性早熟/加速生长

建议：
继续正常发育，无需特殊干预。
```

### 示例 10: 骨龄提前
```
用户: /puberty bone-age 9.5 chronological 8

系统:
骨龄已记录

实际年龄：8.0岁
骨龄：9.5岁
差异：+1.5岁

评估：
骨龄明显提前（>1岁）

风险提示：
可能提示性早熟或加速生长。

建议：
• 咨询儿科内分泌医生
• 进行性激素水平检测
• 监测身高增长速度
• 必要时进行垂体MRI检查
```

### 示例 11: 骨龄延迟
```
用户: /puberty bone-age 13 chronological 15

系统:
骨龄已记录

实际年龄：15.0岁
骨龄：13.0岁
差异：-2.0岁

评估：
骨龄明显延迟（<-1岁）

可能原因：
• 生长激素缺乏
• 甲状腺功能减退
• 遗传因素
• 营养不良
• 慢性疾病

建议：
• 咨询儿科内分泌医生
• 进行激素水平检测
• 评估生长曲线
• 考虑生长激素治疗
```

## Tanner分期速查表

### 女孩乳房发育

| 分期 | 描述 | 年龄范围 |
|:----:|------|---------|
| B1 | 青春期前，无发育 | - |
| B2 | 乳房芽萌出，乳头小隆起 | 8-13岁 |
| B3 | 乳房和乳晕增大，轮廓不清 | 10-14岁 |
| B4 | 乳晕突出，形成第二隆起 | 11-15岁 |
| B5 | 成熟乳房，乳晕回退 | 13-18岁 |

### 男孩睾丸发育

| 分期 | 体积 | 年龄范围 |
|:----:|-----:|---------|
| G1 | <4ml，青春期前 | - |
| G2 | 4-6ml，开始增大 | 9-14岁 |
| G3 | 6-10ml，持续增大 | 10-15岁 |
| G4 | 10-15ml，接近成熟 | 11-16岁 |
| G5 | >=15ml，成熟大小 | 13-18岁 |

### 阴毛发育（男女通用）

| 分期 | 描述 | 年龄范围 |
|:----:|------|---------|
| P1 | 无阴毛，或仅有绒毛 | - |
| P2 | 稀疏、长、色素浅 | 9-16岁 |
| P3 | 变粗、卷曲、色深 | 11-17岁 |
| P4 | 成人型但范围小 | 12-18岁 |
| P5 | 成人型，扩散至大腿内侧 | 13-18岁 |

## 重要时间节点

### 女孩
- 乳房发育开始：8-13岁（平均10岁）
- 阴毛出现：通常在乳房发育后
- 身高增长高峰：乳房发育后1-2年
- 初潮：通常在乳房发育后2-3年（10-15岁）
- 初潮后身高增长：平均5-7cm

### 男孩
- 睾丸增大开始：9-14岁（平均11.5岁）
- 阴毛出现：与睾丸发育同期或稍后
- 身高增长高峰：睾丸发育后1-2年
- 变声：G3-G4期
- 遗精：G4-G5期
- 最终身高：18岁左右
