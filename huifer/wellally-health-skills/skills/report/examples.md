# report 技能示例

## 示例 1: 生成综合报告

```
用户: 生成综合健康报告

输出:
✅ 健康报告已生成

文件路径：reports/health-report-2025-12-31.html
报告类型：综合健康报告
数据范围：所有数据

包含章节：
✓ 患者概况
✓ 生化检查分析（12次检查）
✓ 影像检查汇总（5次检查）
✓ 用药分析（3种药物）
✓ 辐射剂量追踪
✓ 过敏摘要
✓ 症状历史
✓ 手术记录
✓ 出院小结
```

## 示例 2: 生成季度报告

```
用户: 生成综合报告 上季度

输出:
✅ 健康报告已生成
时间范围：2025-10-01 至 2025-12-31
包含章节：9个章节
```

## 示例 3: 生成自定义时间范围报告

```
用户: 生成自定义报告 2024-01-01,2024-12-31

输出:
✅ 健康报告已生成
时间范围：2024-01-01 至 2024-12-31
包含章节：9个章节
```

## 示例 4: 生成特定章节报告

```
用户: 生成自定义报告 2024-01-01,2024-12-31 biochemical,medication,radiation

输出:
✅ 健康报告已生成
时间范围：2024-01-01 至 2024-12-31
包含章节：
• 生化检查分析
• 用药分析
• 辐射剂量追踪
```

## 示例 5: 生成生化趋势分析

```
用户: 生成生化趋势分析 去年

输出:
✅ 健康报告已生成
报告类型：生化趋势分析
时间范围：2024年
图表数量：6个交互式图表
```

## 示例 6: 指定输出文件名

```
用户: 生成综合报告 all all my-health-report.html

输出:
✅ 健康报告已生成
文件路径：reports/my-health-report.html
```
