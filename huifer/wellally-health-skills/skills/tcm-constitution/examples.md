# tcm-constitution 技能示例

## 一、体质辨识评估

### 示例 1: 开始问卷
```
用户: /tcm assess

系统将引导您完成60题问卷：

# 中医体质辨识问卷

说明：
- 共60题，每种体质7-8题
- 请根据最近3个月的情况回答
- 评分标准: 没有(1)/很少(2)/有时(3)/经常(4)/总是(5)

---

### 第1题 (共60题)
**您容易疲乏吗?**

A. 没有 (1分)
B. 很少 (2分)
C. 有时 (3分)
D. 经常 (4分)
E. 总是 (5分)

请输入 A/B/C/D/E:
```

### 示例 2: 问卷完成
```
用户问卷完成后保存:
{
  "primary_type": "气虚质",
  "secondary_types": ["阳虚质"],
  "scores": {
    "peaceful": 42.1,
    "qi_deficiency": 78.5,
    "yang_deficiency": 62.3
  }
}
```

## 三、饮食调养建议

### 示例 4: 气虚质饮食建议
```
用户: /tcm diet 气虚质

输出:
# 气虚质饮食调养建议

## 饮食原则
补气健脾，营养丰富，易于消化

## 宜食食物
- 补气类: 山药、大枣、黄芪、人参
- 健脾类: 薏苡仁、茯苓、扁豆
- 蛋白质: 鸡肉、牛肉、鲫鱼

## 忌食食物
- 生冷寒凉: 冰淇淋、冰镇饮料

## 推荐食谱
1. 黄芪炖鸡
2. 山药粥
3. 红枣茯苓粥

## 饮食建议
- 少食多餐
- 细嚼慢咽
- 饮食温热
- 饭后休息
```

## 六、中药调理建议

### 示例 8: 气虚质中药参考
```
用户: /tcm herbal 气虚质

输出:
⚠️ **重要警告**: 以下内容仅供中医师参考，不可自行抓药服用

## 推荐方剂
**四君子汤加减**

## 方剂组成
| 药材 | 剂量 | 功效 |
|------|------|------|
| 人参 | 9-15g | 大补元气 |
| 白术 | 9-12g | 健脾益气 |
| 茯苓 | 9-15g | 健脾渗湿 |
| 甘草 | 6-9g | 调和诸药 |

## 用法
水煎服，日一剂，分早晚两次温服

## 注意事项
⚠️ **必须遵守**:
- 需经专业中医师辨证后使用
- 孕妇、儿童、体弱者需医师指导
- 服药期间忌食生冷、油腻、辛辣食物
- 感冒发烧时暂停服用
```
