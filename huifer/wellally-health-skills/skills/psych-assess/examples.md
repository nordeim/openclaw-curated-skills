# psych-assess 技能示例

## 示例 1: 开始评估

```
用户: /psych-assess start

系统执行:
1. 显示评估类型选择菜单
2. 用户选择快速筛查或全面评估
3. 开始相应评估流程

输出:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
       心理健康评估 - 选择评估类型
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

请选择适合您的评估类型：

[A] 快速筛查（约2分钟）
    适合：快速情绪检查、日常监测
    内容：PHQ-2 + GAD-2 + 危机检测

[B] 全面评估（约10-15分钟）
    适合：定期体检、症状详查、首次评估
    内容：多维度量表 + 完整评估

[C] 我不太确定，帮我推荐

[D] 先了解什么是心理健康评估
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
