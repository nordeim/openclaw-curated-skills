# Child Safety Assessment Examples

### User Input
```
/child-safety record home
```

### System Output
```

2. Are furniture sharp corners equipped with bumpers?
3. Are windows fitted with guards or limiters?
4. Are cleaning supplies/medications stored out of reach?
5. Is the bathroom fitted with non-slip mats?

Please answer each item...
```

## Example 2: Risk Assessment

### User Input
```
/child-safety risk fall
```

### System Output
```
Fall Risk Assessment

Child: Xiaoming (2 years 5 months)

Risk Level: High Risk

High Risk Scenarios:

1. Window Falls
   Risk Factors: Climbing, no protection
   Prevention:
   Install window guards
   Keep furniture away from windows
   Teach child not to climb

2. Bedside Falls
   Risk Factors: Rolling, climbing
   Prevention:
   Use bed rails
   Lower mattress to lowest setting

3. Stair Falls
   Risk Factors: No safety gates
   Prevention:
   Install safety gates top and bottom
   Teach proper stair climbing

4. Furniture Tip-Over
   Risk Factors: Unstable climbing
   Prevention:
   Secure tall furniture
   Mount TV to wall

Emergency Response:
If a fall occurs:
1. Check consciousness
2. Observe for external injuries
3. Seek medical attention if abnormalities
4. Observe for 24 hours after head injury

Use /child-safety emergency to view first aid information
```

## Example 3: Prevention Recommendations

### User Input
```
/child-safety prevent
```

### System Output
```
Safety Prevention Recommendations

Child: Xiaoming (2 years 5 months)

Home Safety:

Window Safety
  Install guards or limiters (open no more than 10cm)
  Keep tables and chairs away from windows

Fall Protection
  Install corner guards on furniture
  Non-slip flooring

Electrical Safety
  Install protective caps on all outlets
  Secure and hide electrical cords

Burn Prevention
  Keep kettle at the back
  Don't use tablecloths (prevent pulling)
  Run cold water before hot in bath

Poison Prevention
  Store medications in locked high places
  Keep cleaners in original packaging
  Never use beverage bottles for chemicals

Traffic/Car Safety:

Car Seat
  Use rear-facing car seat (until at least 2 years)
  Proper installation, regular checks

Riding Safety
  Don't sit in front passenger seat
  Don't extend body out of windows

Food/Eating Safety:

Choking Prevention
  Avoid whole nuts, grapes, gummies
  No playing/running while eating
  Don't play with chopsticks/toothpicks

Water Safety:

Bathroom Safety
  Bath water depth no more than 5cm
  Never leave sight

Pool/Pond
  Install fence around perimeter
  Don't let child approach alone

Outdoor Safety:

Getting Lost Prevention
  Teach child to remember parents' phone
  Hold hands when going out
  Use location tracking wristband
```
