## 辐射剂量标准值

**CT检查（mSv）：**
- 头部: 2
- 胸部: 7
- 腹部: 8
- 盆腔: 6
- 脊柱: 6
- 全身: 10

**X光检查（mSv）：**
- 胸部: 0.1
- 腹部: 0.7
- 四肢: 0.01
- 牙齿: 0.005

## 剂量调整

```
调整系数 = 实际体表面积 / 1.73
实际剂量 = 标准剂量 × 调整系数
```

## 辐射衰减

```
残留剂量 = 初始剂量 × (0.5)^(经过年数)
```

## 安全阈值

| 状态 | 年剂量 |
|-----|-------|
| 安全 | < 1 mSv |
| 关注 | 1-10 mSv |
| 警告 | 10-50 mSv |
| 危险 | > 50 mSv |
