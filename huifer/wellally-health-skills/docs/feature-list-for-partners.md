## 一、产品概述

WellallyHealth 是一套基于 Claude Code 技术构建的个人健康管理系统，通过 65+ 专业化健康技能模块，提供覆盖全生命周期、全健康场景的智能健康管理能力。

### 核心价值

- **全生命周期覆盖**：从儿童发育到老年认知健康管理
- **智能 AI 分析**：基于 Claude AI 的健康数据分析和风险预测
- **模块化架构**：65+ 独立技能模块，灵活组合部署
- **医学安全管控**：严格的医疗安全红线，不替代专业诊断

## 二、AI 智能健康核心

| 功能模块 | 能力描述 |
|---------|---------|
| **AI 综合分析** | 多维健康数据分析、趋势识别、异常检测、关联分析（Pearson/Spearman相关系数） |
| **健康风险预测** | 高血压/糖尿病/心血管疾病10年风险预测（Framingham模型） |
| **智能健康问答** | 自然语言健康数据查询、趋势分析、关联关系分析 |
| **健康报告生成** | HTML格式综合报告，含ECharts图表可视化 |

### AI 分析能力详解

- **趋势分析**：线性回归、移动平均
- **异常检测**：CUSUM、Z-score 算法
- **个性化建议**：三级健康建议体系
- **报告类型**：综合报告、快速摘要、风险评估、趋势分析

## 三、基础健康档案

| 功能模块 | 能力描述 |
|---------|---------|
| **个人档案管理** | 性别、身高、体重、出生日期，自动计算BMI/BSA |
| **过敏史管理** | 药物/食物/环境过敏记录，严重程度分级，医疗警示集成 |
| **症状记录** | 自然语言症状录入，自动医学术语转换，严重程度评估 |
| **用药管理** | 用药计划、服药记录、依从性追踪，含过敏检查/妊娠安全检查/药物相互作用检查 |

### 数据结构

- 自动计算指标：BMI、体表面积(BSA)、年龄
- 过敏分级：轻度(1级)、中度(2级)、重度(3级)、过敏性休克(4级)
- 用药依从性：优秀(≥90%)、良好(70-89%)、需改进(<70%)

## 四、慢病管理

| 功能模块 | 能力描述 |
|---------|---------|
| **糖尿病管理** | 血糖监测、HbA1c追踪、TIR分析、低血糖事件记录、并发症筛查 |
| **高血压管理** | 血压监测、靶器官损害评估、心血管风险评估、用药追踪 |
| **COPD管理** | 肺功能监测(FEV1)、CAT评分、mMRC评估、急性加重追踪 |
| **认知功能评估** | MMSE/MoCA测试、认知域评估、ADL/IADL评估 |

### 糖尿病管理

- **血糖类型**：空腹血糖、餐后2小时血糖、睡前血糖、随机血糖
- **目标范围**：空腹 4.4-7.0 mmol/L，餐后2h <10.0 mmol/L
- **TIR 分析**：目标范围内时间占比，目标 >70%
- **并发症筛查**：视网膜病变、糖尿病肾病、神经病变、足部并发症

### 高血压管理

- **血压分级**：正常、正常高值、1级高血压、2级高血压、3级高血压
- **靶器官评估**：心脏、肾脏、视网膜、动脉僵硬度
- **心血管风险**：ASCVD 10年风险评估

## 五、女性健康管理

| 功能模块 | 能力描述 |
|---------|---------|
| **月经周期追踪** | 周期记录、PMS症状、排卵预测、健康洞察 |
| **孕期管理** | 产检记录、症状追踪、体重管理、胎动监测、多胎妊娠支持 |
| **产后恢复** | 恶露记录、疼痛管理、母乳喂养、EPDS抑郁筛查、盆底肌恢复 |
| **更年期管理** | 症状追踪、HRT治疗、骨密度监测、心血管风险评估 |

### 孕期管理

- **产检计划**：NT扫描(12周)、唐筛/NIPT(16周)、大排畸(20周)、糖耐量(24周)
- **多胎支持**：双胎/多胎妊娠管理、预产期自动调整、产检频率优化
- **营养建议**：叶酸、铁、钙、DHA 补充建议

### 更年期管理

- **分期识别**：围绝经期、绝经期、绝经后期
- **症状评分**：潮热、盗汗、睡眠、情绪等综合评分
- **骨密度**：T-score 评估、骨折风险分层

## 六、儿童健康管理

| 功能模块 | 能力描述 |
|---------|---------|
| **发育里程碑** | ASQ-3/Denver II标准，5大发育域评估，发育迟缓预警 |
| **儿童疾病** | 发热/咳嗽/腹泻等常见症状记录，用药记录 |
| **儿童心理** | 行为评估、情绪追踪、ADHD筛查 |
| **儿童营养** | 饮食记录、营养评估、挑食管理 |
| **儿童安全** | 意故伤害预防评估、家庭安全检查 |
| **儿童睡眠** | 睡眠模式追踪、睡眠问题识别 |
| **疫苗接种** | 疫苗计划、接种记录、提醒功能 |

### 发育评估

- **五大发育域**：大运动、精细运动、语言、社交、认知
- **评估标准**：ASQ-3、Denver II
- **早产校正**：胎龄<37周者自动校正至2岁

## 七、健康管理

| 功能模块 | 能力描述 |
|---------|---------|
| **营养追踪** | 拍照记录饮食，自动营养成分分析（热量、蛋白质、脂肪、碳水、维生素、矿物质） |
| **运动健身** | 运动记录、健身目标管理、运动分析、FITT原则参考 |
| **睡眠管理** | 睡眠记录、PSQI/Epworth/ISI评估、睡眠问题识别 |
| **心理健康** | PHQ-9/GAD-7/PSQI量表、情绪日记、治疗记录、危机管理计划 |

### 营养分析

- **自动识别**：食物类型、份量估算、烹饪方式
- **营养成分**：宏量营养素、维生素、矿物质
- **健康评分**：综合营养评分、平衡度、多样性

### 运动管理

- **运动类型**：有氧运动、力量训练、球类运动、其他运动
- **强度评估**：RPE主观疲劳度、心率区间
- **目标类型**：减重、增肌、耐力、健康、习惯养成

## 八、专科健康监测

| 功能模块 | 能力描述 |
|---------|---------|
| **眼科健康** | 视力记录、眼压监测、眼底检查、青光眼/白内障/AMD筛查 |
| **跌倒风险评估** | 跌倒史记录、TUG/Berg/单腿站立测试、步态分析、居家安全评估 |
| **辐射剂量管理** | 医疗辐射累积剂量追踪 |
| **康复管理** | 康复计划、进展追踪 |

### 眼科健康

- **视力分级**：正常(1.0-1.5)、轻度下降(0.8-0.9)、中度下降(0.4-0.7)、重度下降(<0.1)
- **眼压参考**：正常范围 10-21 mmHg
- **检查频率建议**：成人常规、高危人群年度检查

### 跌倒风险

- **平衡测试**：TUG计时起立行走测试、Berg平衡量表、单腿站立测试
- **风险因素**：年龄、跌倒史、平衡功能障碍、步态异常等10项评估
- **环境评估**：客厅、卧室、浴室、楼梯安全检查

## 九、医疗服务支持

| 功能模块 | 能力描述 |
|---------|---------|
| **多学科会诊(MDT)** | 协调专家团队分析医疗数据，生成综合诊断报告 |
| **医疗记录查询** | 生化检验/影像检查查询、异常指标筛选 |
| **健康报告导出** | 自定义HTML报告，支持多种图表可视化 |
| **出院管理** | 出院小结记录、康复计划追踪 |

### MDT 会诊

- **支持专科**：心内科、内分泌科、消化科、肾内科、血液科、呼吸科、神经内科、肿瘤科、全科
- **智能分诊**：根据异常指标自动识别需求专科
- **报告结构**：各专科分析、综合评估、优先级排序、综合建议

## 十、技术特性

### 数据存储

| 特性 | 说明 |
|-----|------|
| **数据来源** | 用户主动上传或录入，系统不自动采集任何个人健康信息 |
| **数据格式** | JSON 标准化存储 |
| **组织方式** | 按日期分层组织 (YYYY-MM/YYYY-MM-DD) |
| **索引管理** | 全局索引文件支持快速查询 |
| **支持上传** | 检验报告、影像报告、症状记录、用药记录、饮食照片等 |

### 安全管控

| 特性 | 说明 |
|-----|------|
| **医疗红线** | 不诊断、不开处方、不预测预后、不替代医生 |
| **数据隐私** | 本地优先存储，用户完全控制 |
| **紧急预警** | 风险阈值 0.7，自动触发就医建议 |

### 技术架构

| 特性 | 说明 |
|-----|------|
| **技能标准** | Agent Skills 开放标准 |
| **技能数量** | 65+ 独立技能模块 |
| **可扩展性** | 模块化设计，支持技能热插拔 |
| **自然语言** | 中文自然语言输入，自动医学标准化 |
