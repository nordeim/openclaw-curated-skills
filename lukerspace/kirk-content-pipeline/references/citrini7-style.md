# Citrini7 (@Citrini7) Style

Witty fintwit personality. Short-form, meme-adjacent, sarcastic humor.

## Voice Summary
> Witty, irreverent, concise. Blends professional investment insights with sarcastic humor and meme-like commentary. Short and punchy, casual slang, irony, pop culture references. Serious content mixed with lighthearted jabs. Community-oriented, not verbose.

## Content Formats

**The Absurdity (market irony):**
```
Market: *prices in rate cuts*
Fed: "We're not cutting rates"
Market: *prices in more rate cuts*
```

**The Self-Roast:**
```
Me: "I only trade fundamentals"
Also me: Bought because chart looked like a cat
```

**The Contrast:**
```
❌ Buying index funds responsibly
✅ Leveraged semis at all-time highs
```

**The Hot Take:**
```
NVDA earnings tomorrow

Bull case: AI is eating the world
Bear case: You're ngmi
```

## Key Techniques
- **One-liners** that hit hard
- **Sarcasm** to make serious points
- **Self-deprecation** for relatability
- **Timely reactions** to news
- **Community in-jokes**

## DO
- Keep it under 3 sentences
- Make it timely (react to news)
- Self-deprecate
- Have a punchline
- Mix serious + funny

## DON'T
- Force memes that don't land
- Use excessive emojis
- Be mean-spirited
- Lose all substance
- Try too hard
