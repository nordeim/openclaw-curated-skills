# 使用時機
當使用者說「幫我做個案紀錄 / 產生紀錄 / 把這段LINE整理成紀錄 / 用路線B處理」時，套用本技能。

# 0) 資料最小化（隱私）
- 不輸出真實姓名、電話、地址、身分證等
- 人名一律用代號：案主A、鄰居B、家屬C
- 引言最多 1 句，且避開敏感細節

# 1) 先抽取關鍵資訊（從聊天內容自動抓）
- 主題/事件：例如「鄰居說壞話（指控：很吵）」
- 案主表達：情緒、困擾點
- 目標：降低衝突、澄清誤會、建立界線
- 已選路線：A低衝突 / B正式溝通 / C留證據走管道
- 具體指控類型：例如「很吵」
- 可行承諾：例如「晚上10點後注意音量、放輕腳步」

# 2) 個案紀錄固定輸出格式（預設：社工敘事 + 介入計畫）
【個案紀錄】
- 日期時間：
- 來源/管道：LINE/口述/面談
- 案主代號：
- 主題：
- 問題陳述（S）：
- 客觀資訊（O）：
- 初步評估（A）：
- 介入措施（I）（本次提供了什麼）：
- 處理路線：A/B/C（若未指明，預設先給A/B/C選單）
- 建議話術（路線B優先）：
- 風險與注意事項：
- 後續追蹤計畫（P）：

# 3) 路線模板（可直接複用）
## 路線A：低衝突止血（簡短、避衝突）
- 先降噪/調整時段/避免正面衝突
- 目標：先讓生活恢復可控

## 路線B：正式溝通（講清楚但不撕破臉）
(1) 時機：白天、走廊遇到、彼此不趕時間；避免在眾人旁
(2) 開場一句：
「我想跟你好好講一件事，不吵架，花你兩分鐘就好。」
(3) 三段式（觀察感受要求）：
「我有聽到有人轉述你提到一些關於我的負面說法。
我聽到會不舒服，也擔心影響彼此相處。
我希望以後如果你對我有意見，請直接跟我說，不要在外面聊我的事。」
(4) 立界線：
「你如果覺得我哪裡不對，我願意聽也願意改；但背後講我不接受。」
(5) 否認/反咬應對：用一句話回到界線與就事論事
(6) 收尾：
「謝謝你聽我說，我希望之後都能好好做鄰居。」

## 路線B-精準版（針對指控：很吵）
「我有聽到你說我很吵。如果真的影響到你，我願意改善。
但我希望你以後覺得我吵，請直接跟我說，不要在外面跟別人講，避免誤會傷感情。
我想確認：你覺得哪個時段吵？是腳步聲/講話/電視/音樂/搬東西？
你講清楚我才好調整。
我先做到：晚上10點後特別注意音量、放輕腳步。」

## 路線C：留證據＋走管道
- 若持續被抹黑/衝突升高：紀錄時間地點、保留訊息、必要時走管委會/房東/正式申訴
- 目標：自保、降風險、可被第三方理解的證據鏈

# 4) 生成規則
- 先輸出「個案紀錄」再輸出「可直接複製的話術」
- 不確定欄位用「待補」標註，最多問 2 個補充問題
