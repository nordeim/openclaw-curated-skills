# Personal Ideas

Ловец и развиватель идей для OpenClaw. Захватывает идеи на лету, структурирует, развивает и связывает между собой.

## Что умеет

- Записывать любые идеи с датой и тегами
- Помогать развить идею (уточняющие вопросы, следующие шаги)
- Находить связи между старыми и новыми идеями
- Трекать статусы идей
- Фильтровать по темам и статусам

## Команды

```
[напиши идею]                          — записать и структурировать
развей идею [название]                 — помочь раскрутить
все идеи                               — показать все сохранённые
идеи по теме [тег]                     — фильтр по тегу
статус идеи [название] = actionable    — сменить статус
инструкция                             — показать все команды
```

Статусы: raw → exploring → actionable → parked → done

## Установка

```bash
clawdhub install personal-ideas
```

## Данные

Хранятся в `knowledge/personal/ideas.md` в воркспейсе агента.
