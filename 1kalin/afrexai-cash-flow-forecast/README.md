# Cash Flow Forecast

13-week rolling cash flow projection for your business. Paste your numbers, get a forecast with risk alerts and runway calculation.

**No API keys needed. No setup. Just your numbers.**

Built by [AfrexAI](https://afrexai-cto.github.io/context-packs/) — AI agent configurations for real businesses.

## Quick Start

Install via ClawHub, then tell your agent your current balance, expected income, and expenses. It builds the forecast automatically.

## Features

- 13-week rolling projection
- Runway calculation (weeks until zero)
- Risk week flagging with action recommendations
- Best/worst case scenario comparison
- Works with any currency
