# Daily Briefing

Creates a concise morning briefing with your schedule, priorities, inbox highlights, and relevant news. Start every day knowing what matters.

## What It Does

- Pulls calendar events and flags upcoming meetings
- Highlights urgent emails and follow-ups due
- Prioritizes your day using an importance/urgency matrix
- Includes relevant industry news
- Optional end-of-day recap

## Usage

"Brief me." That's it. Or set it up on a morning cron job for automatic delivery.
