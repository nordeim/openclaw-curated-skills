# OKR Framework — Objectives & Key Results Planner

Structured OKR planning for companies, teams, and individuals. Write better goals, score them honestly, and run quarterly retrospectives that actually improve performance.

## Features
- Company → Team → Individual OKR cascade
- 0.0-1.0 scoring with calibration guidelines
- Quarterly review templates
- AI agent OKR patterns (treat agents like employees)
- Industry-specific templates (SaaS, Healthcare, Fintech, Manufacturing, Professional Services)
- CFR companion (Conversations, Feedback, Recognition)

## Usage
Ask your agent: "Help me set OKRs for Q2" or "Create OKRs for my engineering team"
