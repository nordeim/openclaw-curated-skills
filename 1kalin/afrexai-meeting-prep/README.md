# Meeting Prep

Prepares briefing docs before your meetings so you walk in ready. Researches the people, the company, and suggests talking points.

## What It Does

- Researches attendees (LinkedIn, recent posts, background)
- Pulls company intel and recent news
- Suggests talking points and smart questions
- Delivers a scannable one-pager briefing
