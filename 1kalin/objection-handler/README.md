# Objection Handler

Handles sales objections with proven response frameworks. Covers price, timing, trust, status quo, and brush-off objections.

## What It Does

- Uses the LAER framework (Listen, Acknowledge, Explore, Respond)
- Covers 15+ common objections across 5 categories
- Finds the real concern behind the stated objection
- Provides follow-up questions and multiple response options

## Usage

"The prospect said it's too expensive. How do I handle that?"
"They want to wait until next quarter. What do I say?"
