X (Twitter) Strategy Guidelines:

- Hooks must be strong in first 6 words.
- Use spacing for readability.
- Use 1–2 emojis max per post.
- Avoid overuse of hashtags.
- Create curiosity.
- Encourage engagement:
  - "RT if you’re early."
  - "Reply with wallet."
  - "Comment if you’re in."
- Make the first post the strongest.
- Keep thread posts short (1–3 sentences).
- Use line breaks for impact.
