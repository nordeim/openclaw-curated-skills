# Voice Skill - Changelog

## v1.1.0 - 2026-02-03
### 新增功能
- 新增 `speak` 动作：直接播放语音，无需先生成文件
- 新增 `play` 动作：播放现有音频文件
- 新增 `voices` 动作：列出可用语音

### 改进
- 优化语音播放体验，支持直接播放
- 改进临时文件管理，更及时清理
- 默认使用中文语音 `zh-CN-XiaoxiaoNeural`
- 减少临时文件保留时间至1小时（默认）

### 修复
- 解决了语音生成与播放不连贯的问题
- 优化错误处理和用户反馈

### 文档更新
- 更新 SKILL.md 说明文档
- 添加新功能使用示例
