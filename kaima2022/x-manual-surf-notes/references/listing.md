# ClawHub listing text (bilingual)

## 中文简介
`x-manual-surf-notes` 是一个“手动刷 X 并做笔记”的工作流技能：通过 OpenClaw 的 Chrome Extension Relay 控制你现有的 Chrome 标签页，在 X 首页 For You 流里像人一样下滑加载、点进帖子详情页，把推文内容用中文转述，并去重后按固定模板追加到笔记文档（时间｜内容｜链接｜评论）。

适合：实时跟 AI / 工具 / 产品前沿，但不想用 API 爬取。

## English
`x-manual-surf-notes` is a manual, browser-driven X (Twitter) surfing workflow skill. It uses OpenClaw’s Chrome Extension Relay to control your existing Chrome tab, scroll the X Home “For You” feed, open tweet detail pages, translate/summarize tweets into Chinese, deduplicate, and append structured notes (Time | Content | Link | Comment) to a markdown file.

Best for: keeping up with AI/frontier updates while explicitly avoiding API scraping.
