### 2.3 A/B Testing Framework

**What to A/B Test (Priority Order)**:

| Element | Impact Potential | Test Ideas |
|---------|------------------|------------|
| Offer Type | Very High | % off vs $ off vs free shipping |
| Headline | High | Question vs statement, urgency |
| CTA Button | High | Color, text, size |
| Timing | Medium | 5s vs 10s vs 15s |
| Design | Medium | Modal vs slide-in vs bar |
| Form Fields | Medium | Email only vs email + name |
| Images | Medium | Product vs lifestyle vs none |

**A/B Testing Rules**:

```
1. Test ONE variable at a time
2. Run for minimum 7 days or 1,000 views
3. Aim for 95% statistical significance
4. Document all tests and results
5. Winner becomes new control
6. Continuously test (never stop)
```

**Common A/B Test Results**:

| Test | Typical Winner | Lift |
|------|---------------|------|
| 10% off vs 15% off | 15% off | +23% |
| % off vs $ off | Depends on price point | Varies |
| Free shipping vs % off | Free shipping | +31% |
| Modal vs slide-in | Modal (for email) | +18% |
| Single field vs multi-field | Single field | +25% |
| Timer vs no timer | With timer | +15% |

### 2.5 Pricing Psychology

**Pricing Tactics That Work**:

| Tactic | Example | Psychology |
|--------|---------|-----------|

**Discount Psychology**:

```
Rule: % off vs $ off

Price < $100: Use % off (sounds bigger)
  "$30 item: Save 20%" beats "Save $6"

Price > $100: Use $ off (sounds bigger)
  "$500 item: Save $100" beats "Save 20%"

Always test both for your audience!
```
