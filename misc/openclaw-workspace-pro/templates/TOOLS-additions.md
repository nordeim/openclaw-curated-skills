

### Network Security - Approved Domains
**Only access these domains without asking first:**
- `*.googleapis.com`, `oauth2.googleapis.com` (Google APIs)
- `api.brave.com` (search)
- `api.telegram.org` (messaging)
- `github.com`, `*.github.com` (code repos)
- `clawhub.com`, `*.clawhub.com` (skills)
- Standard package managers (Homebrew, npm, pip official registries)

**For new domains:** Ask first with brief reason.

**Never access without security review:**
- Pastebin-type services (data exfiltration risk)
- Ngrok/remote shells
- Anonymous/temporary services
