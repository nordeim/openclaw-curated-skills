## 查询方法

未知车站代码时，使用 mcporter 查询：

```bash
# 按城市查询
mcporter call 12306.get-station-code-of-citys citys="城市名"

# 按车站名查询
mcporter call 12306.get-station-code-by-names stationNames="车站名1|车站名2"
```
