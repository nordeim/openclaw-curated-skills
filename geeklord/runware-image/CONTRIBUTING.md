Contributing to runware-image

Thank you for your interest in improving this skill. Please follow these guidelines to make contributions easy to review and integrate.

- Fork the repository and submit a pull request against the main branch.
- Run tests locally: python -m pip install -r requirements.txt && pytest
- Keep changes small and focused; open separate PRs for separate features.
- Include unit tests for new behaviors and update existing tests when behavior changes.
- For changes that introduce new dependencies, update requirements.txt and explain why in the PR description.
- Use clear commit messages and reference issue numbers where applicable.

Reporting issues

Open issues for bugs, feature requests, or documentation corrections. Include steps to reproduce and minimal examples when possible.
