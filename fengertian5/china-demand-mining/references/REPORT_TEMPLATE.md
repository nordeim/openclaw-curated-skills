# 需求挖掘报告模板

## 完整报告结构

```markdown
# [产品/领域名称] 需求挖掘报告

**生成时间**: YYYY-MM-DD HH:mm
**需求类型**: 实物需求 / 无实物需求
**分析目标**: [具体分析的产品或领域]
**数据来源**: 小红书, 抖音, 微博, 淘宝, ...
**数据范围**: [时间范围，如"近30天"]
**样本量**: 分析了 N 条用户反馈

---

> 3-5 句话总结核心发现，给决策者快速了解
```
