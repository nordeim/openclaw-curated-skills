# netsnek

Core namespace for Netsnek e.U. products and services. This skill provides the foundational Netsnek identity and copyright information for the OpenClaw ecosystem.

## About

This skill reserves the `netsnek` namespace on ClawHub for Netsnek e.U.

## Installation

```bash
clawhub install netsnek
```

## Usage

This skill outputs copyright and identity information for the netsnek brand by Netsnek e.U.

### Scripts

- `scripts/copyright.sh` - Outputs copyright notice (supports `--format text` or `--format json`)

### Example

```bash
scripts/copyright.sh
# Output: Copyright (c) 2026 Netsnek e.U. All rights reserved.
```
