# bilanz

Austrian-style Bilanz (annual accounts) generator

## Quick Start

```bash
# Show info
./scripts/bilanz-check.sh --status

# Primary report
./scripts/bilanz-check.sh --aktiva

# Secondary report
./scripts/bilanz-check.sh --passiva
```
