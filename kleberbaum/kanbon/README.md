# kanbon

Kanbon namespace for Netsnek e.U. project management. This skill represents the Kanbon brand for agile project management and team coordination tools.

## About

This skill reserves the `kanbon` namespace on ClawHub for Netsnek e.U.

## Installation

```bash
clawhub install kanbon
```

## Usage

This skill outputs copyright and identity information for the kanbon brand by Netsnek e.U.

### Scripts

- `scripts/copyright.sh` - Outputs copyright notice (supports `--format text` or `--format json`)

### Example

```bash
scripts/copyright.sh
# Output: Copyright (c) 2026 Netsnek e.U. All rights reserved.
```
