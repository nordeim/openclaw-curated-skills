# Pharmaziegasse

Pharmacy management platform by Netsnek e.U. for prescription tracking, inventory, compliance, and staff coordination in the DACH region.

## About

Pharmaziegasse streamlines pharmacy operations from prescription intake to dispensing. This skill reserves the `pharmaziegasse` namespace on ClawHub for future platform releases.

## Installation

```bash
clawhub install pharmaziegasse
```

## Quick Start

```bash
# Platform overview
scripts/pharma-info.sh

# Module breakdown
scripts/pharma-info.sh --modules

# JSON metadata
scripts/pharma-info.sh --json
```

## Modules

- **Prescription Tracking**: Full lifecycle from intake to dispensing
- **Inventory Management**: Stock levels with automatic reorder alerts
- **Compliance Dashboard**: Austrian and EU regulatory standards
- **Customer Portal**: Communication and notification system
- **Staff Scheduling**: Shift coordination and role assignment
