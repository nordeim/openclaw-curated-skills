# 风格润色器

## 技能信息
- **技能名称**：风格润色器
- **技能描述**：润色用户发送的消息，支持多种预置风格和自定义风格
- **适用场景**：日常聊天、社交媒体发布、邮件撰写、内容创作
- **不适用场景**：需要保持原始语义的专业文档、法律文件

## 支持的风格
1. **锐评（辛辣/毒舌）**：使用辛辣的语言，带有讽刺和调侃的语气
2. **二次元（日式动漫梗及口癖）**：使用动漫风格的语言，包含日式口癖和网络用语
3. **古风（文言文/古风小生）**：使用文言文风格的语言，模仿古代文人的说话方式
4. **霸总（油腻短剧霸总）**：使用霸道总裁风格的语言，带有强势和油腻的特点
5. **正式（严肃/官方/商务）**：使用正式、严肃的语言，适合商务和官方场合
6. **自定义风格**：根据用户的具体要求进行个性化润色

## 使用方法
### 方法1：标准格式
```
风格：[风格名称]，内容：[需要润色的文本]
```

### 方法2：简化格式
```
[风格名称] [需要润色的文本]
```

### 方法3：自定义风格
```
风格：[自定义风格描述]，内容：[需要润色的文本]
```

## 示例
### 示例1：锐评风格
**输入**：风格：锐评，内容：今天天气真好
**输出**：这话说得跟没说似的，今天天气真好？也就那样吧

### 示例2：二次元风格
**输入**：二次元 今天天气真好
**输出**：今天天气真好的说～☆

### 示例3：古风风格
**输入**：古风 今天天气真好
**输出**：今日天气甚善矣

### 示例4：霸总风格
**输入**：霸总 今天天气真好
**输出**：女人，你成功引起了我的注意，今天天气真好...（邪魅一笑）

### 示例5：正式风格
**输入**：正式 今天天气真好
**输出**：今日天气良好，谢谢。

### 示例6：自定义风格
**输入**：风格：可爱萌妹，内容：今天天气真好
**输出**：[可爱萌妹] 今天天气真好～

## 注意事项
1. **语义保持**：润色过程中会尽量保持原始文本的语义，但某些风格可能会对语义进行适当调整
2. **风格一致性**：同一文本使用不同风格润色会产生完全不同的效果
3. **适用场景**：请根据具体使用场景选择合适的风格
4. **自定义风格**：自定义风格的效果可能会因描述的详细程度而有所不同
5. **语言限制**：目前主要支持中文文本的润色

## 常见问题
### Q: 润色后的文本和原始文本差异很大怎么办？
**A**: 这是正常现象，不同风格会对文本进行不同程度的改写。如果需要更接近原始文本的润色，可以选择正式风格。

### Q: 自定义风格的效果不理想怎么办？
**A**: 请尝试更详细地描述您想要的风格特点，例如"可爱萌妹风格，使用更多语气词和表情"。

### Q: 可以润色英文文本吗？
**A**: 目前主要针对中文文本进行优化，英文文本的润色效果可能会有所折扣。
