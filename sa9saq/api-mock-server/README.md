```
"Start a mock server from my openapi.yaml"
"Create a mock API with GET /users and POST /orders"
"Mock this Swagger spec: https://petstore.swagger.io/v2/swagger.json"
```

- Node.js (for npx)
- No additional installs needed (prism runs via npx)
