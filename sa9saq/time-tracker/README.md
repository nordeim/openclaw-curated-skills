# ⏱️ time-tracker

File-based work time tracking and reporting. No external services needed.

## Features

- Start/stop timers per project and task
- JSONL storage (simple, portable, git-friendly)
- Daily, weekly, and monthly reports
- Project-level aggregation and summaries
- Current timer status check
- No database or external service required

## Usage

```
"Start tracking time on project-x"
"Stop timer"
"Show my time report for this week"
"How much time did I spend on project-x this month?"
"List all tracked projects"
```

## Requirements

- Shell access for file operations
- Data stored in `~/.time-tracker/`
