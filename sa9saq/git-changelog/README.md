Auto-generate beautiful changelogs from your git history.

## Features

- Parses Conventional Commits (`feat:`, `fix:`, `docs:`, etc.)
- Groups changes by category with emoji headers
- Detects version from git tags
- Links commits to remote repository
- Supports custom date ranges and tag ranges
- Falls back gracefully for non-conventional commit styles

## Usage Examples

- "Generate a changelog for this repo"
- "Create release notes from v1.0.0 to v2.0.0"
- "Summarize git commits from the last 2 weeks"

## Requirements

- `git` installed and run inside a git repository
- Shell access
