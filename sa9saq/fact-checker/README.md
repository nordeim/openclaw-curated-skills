Automated fact-checking skill that verifies claims using web search and assigns confidence scores.

## Features

- Extracts and isolates individual claims from text
- Searches multiple sources for verification
- Assigns confidence scores (Verified / Partially True / False / Unverifiable)
- Provides source URLs for independent verification
- Generates structured fact-check reports

## Usage Examples

- "Fact-check this article about climate change statistics"
- "Verify the claims in this press release"
- "Is it true that Python is the most popular programming language in 2025?"

## Requirements

- `web_search` tool access
