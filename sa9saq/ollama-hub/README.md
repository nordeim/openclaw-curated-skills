Manage, list, benchmark, and switch between local Ollama models from your agent.

## Features

- List all installed models with size and quantization info
- Pull new models and remove unused ones
- Benchmark models with standardized prompts
- Side-by-side model comparison
- Show detailed model metadata

## Usage

```
"List my Ollama models"
"Benchmark llama3:8b vs mistral:7b"
"Pull gemma2:9b"
"Show details for codellama:13b"
```

## Requirements

- [Ollama](https://ollama.com) installed and running
- Shell access
