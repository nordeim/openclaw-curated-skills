# Git Hooks

A ClawHub skill for automatically setting up Git hooks in any repository.

## Features

- Pre-commit hooks (linting, formatting)
- Pre-push hooks (test runner)
- Commit-msg hooks (conventional commits enforcement)
- No dependencies beyond Git and shell

## Example Prompts

- "Add a pre-commit hook to lint my JavaScript"
- "Set up conventional commit message validation"
- "Add a pre-push hook that runs tests"
