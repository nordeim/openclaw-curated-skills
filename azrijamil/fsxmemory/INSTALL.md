# Foresigxt Memory Skill Installation

## Quick Install

```bash
npm install -g @foresigxt/foresigxt-cli-memory
```

## Files Included

- **SKILL.md** - Complete skill documentation and reference
- **templates/** - 9 memory templates for structured documentation
- **.env.example** - Configuration template

## Setup

1. Install the package globally (see above)
2. Copy .env.example to .env and configure your vault path
3. Review SKILL.md for complete usage guide
4. Start using: `fsxmemory init ~/memory`

## Documentation

See SKILL.md for:
- All commands and usage
- Configuration options (.env, export, --vault)
- Migration from OpenClaw/Obsidian
- Templates reference
- Context death resilience features
